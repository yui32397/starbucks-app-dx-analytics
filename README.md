# Starbucks App DX Analytics: デジタル会員向け物販の成長要因分解 ＆ 統計的A/Bテスト

東証プライム等の小売・EC市場のベンチマークに準拠した、スターバックス（スタバアプリEC）におけるデジタル会員向け物販（オンラインストア）の売上データおよびユーザー行動ログを活用したデータアナリティクス・プロジェクトです。

実務における「アプリDX推進」「デジタルマーケティングの意思決定」を想定し、数式に基づく厳密な売上要因分解と、統計学（カイ二乗検定）を用いたA/Bテストの有意差判定を実装しています。
